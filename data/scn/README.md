## 参与简体中文KC3改翻译须知

欢迎参与KC3改的简体中文翻译！希望在贡献翻译时尽量依据以下规则：

* 版面宽松的情况下尽量使用全角标点符号
* 装备中的括号使用半角
* 使用Tab缩进
* 以下情况可以直接push至repo：
    * 翻译缺失项目（包括将现存英文条目翻译中文）
    * 修正笔误 / 调整缩进
* 修改已有翻译超过三条请尽量发Pull Request 并 @ 其他scn翻译者 （目前主要活跃的翻译者为： @Javran @Diablohu ）
* 其他情况下自行判断
